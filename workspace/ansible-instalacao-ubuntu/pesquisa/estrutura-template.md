# Estrutura do Template de Labs

## Formato Padrão

1. **Título**: `# Lab XX – [Verbo + o que será feito]`
2. **Objetivo**: 3 a 6 bullets com verbos no infinitivo, sempre incluir limpeza
3. **Pré-requisitos**: ferramentas com links + comandos de instalação em bash
4. **Estrutura de Diretórios**: se aplicável
5. **Passo a Passo**: 5 a 12 passos numerados, separados por `---`
6. **Resultado Esperado**: 3 a 5 resultados concretos
7. **Dicas para Discussão**: tabela (opcional)

## Regras

1. **Passos**: 5 a 12, numerados, separados por `---`
2. **Código**: sempre completo, linguagem correta (bash/yaml/hcl/python/json)
3. **Placeholders**: usar `<VALOR>` para dados que o aluno substitui
4. **Independência**: cada lab executável isoladamente
5. **Progressão**: Lab 01 = setup
6. **LABS DIRETOS**: foco na prática, notas concisas, sem rodeios
7. **Limpeza obrigatória** ao final
